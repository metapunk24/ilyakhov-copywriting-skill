# Чек-лист проверки текста

Пройдите по всем пунктам перед сдачей текста.

---

## Глава 1. Отжать воду

### Вводные слова
- [ ] Убраны «в связи с», «следует отметить», «как известно», «в настоящее время», «в целях», «на основании», «таким образом», «в соответствии с», «убедительно просим», «по имеющейся информации»
- [ ] Заменены на факты или прямые конструкции

### Оценки
- [ ] Убраны «высококачественный», «уникальный», «лучший», «замечательный», «эксклюзивный», «профессиональный», «непревзойдённый», «широкий ассортимент», «высокий уровень», «индивидуальный подход»
- [ ] Заменены на конкретные характеристики и цифры

### Штампы
- [ ] Убраны «не оставляет равнодушным», «обращаем внимание», «по вопросам», «рады сообщить», «на протяжении многих лет», «всех без исключения»
- [ ] Переписаны простыми словами

### Заумное
- [ ] Заменены «осуществлять» → «делать», «проводить работы» → «работать», «оказывать содействие» → «помогать», «принимать участие» → «участвовать»
- [ ] Убраны «инновационные решения», «эксклюзивные коммуникационные проекты», «комплексный подход»

### Эвфемизмы
- [ ] «Оптимизация структуры занятости» → увольнения
- [ ] «Временные трудности» → кризис / проблема
- [ ] Названы вещи своими именами

### Отглагольные существительные
- [ ] «Проведение» → проводим
- [ ] «Осуществление» → делаем
- [ ] «Выполнение» → выполняем
- [ ] «Реализация» → реализуем

### Неопределённость
- [ ] «В ближайшее время» → конкретная дата
- [ ] «В ряде случаев» → конкретный процент или число
- [ ] «По мере необходимости» → конкретное условие

### Брехня без фактов
- [ ] «Всё больше» → данные опроса или статистика
- [ ] «Становится модным» → цифры продаж
- [ ] Или удалено полностью

### Культ слов
- [ ] «Ваш», «Клиент» с заглавной — исправлены на строчные
- [ ] «Прошу рассмотреть» — убрано
- [ ] Текст проверен: звучит как разговор, а не как официальное письмо

## Глава 2. Донести мысль

### Информативность
- [ ] Один абзац — одна мысль
- [ ] Нет абзацев из 10+ предложений
- [ ] Нет односложных абзацев подряд (исключение: акцент)

### Синтаксис
- [ ] Предложения до 20 слов (допустимо до 30)
- [ ] Активный залог вместо пассивного
- [ ] Прямой порядок слов
- [ ] Убраны причастные обороты
- [ ] Убраны деепричастные обороты (где мешают)

### Абзацы
- [ ] 2–5 предложений на абзац
- [ ] Первая фраза — суть абзаца
- [ ] Логическая связь между абзацами

### Структура
- [ ] Пирамида: главное первым
- [ ] Заголовок обещает пользу или новость
- [ ] Лид — главная мысль в 1–2 предложения
- [ ] Есть призыв к действию или вывод

### Заголовок
- [ ] Конкретный (с цифрами, фактами, сроками)
- [ ] Без штампов: «новости», «о нас», «информация», «услуги»
- [ ] Обещает пользу читателю

## Глава 3. Рассказать о себе (если реклама / продажи)

### Решение о покупке
- [ ] Текст решает проблему клиента, а не хвалит продукт
- [ ] Есть описание проблемы
- [ ] Есть решение
- [ ] Есть доказательства

### Продукт
- [ ] Факты вместо оценок
- [ ] Демонстрация вместо заявлений
- [ ] Цифры и примеры

### Компания
- [ ] Убрано «ведущий поставщик решений»
- [ ] Убрано «высокопрофессиональные сотрудники»
- [ ] Конкретно: чем занимаемся, для кого, какой результат

### Регалии
- [ ] Не перечислены награды
- [ ] Показан результат для клиента

### Честность
- [ ] Если есть проблема — сказано прямо
- [ ] Нет прикрашивания и эвфемизмов

---

## 10 правил сильного текста

- [ ] 1. Сила — в правде (нет лжи и притворства)
- [ ] 2. Смысл важнее формы (нет красивых пустых фраз)
- [ ] 3. Чем проще — тем лучше (нет сложных конструкций)
- [ ] 4. Пишите, как для себя (звучит как разговор)
- [ ] 5. Читайте вслух (не задыхаетесь, не скучаете)
- [ ] 6. Приводите примеры (абстракции заменены конкретикой)
- [ ] 7. От простого к сложному (база перед деталями)
- [ ] 8. Пользу — вперёд (уже в заголовке)
- [ ] 9. Поставьте заголовок (обещает конкретную выгоду)
- [ ] 10. Уважение и забота (читателю легко и понятно)

---

## Финальная проверка

- [ ] Прочитано вслух
- [ ] Смысл не потерян при сокращении
- [ ] Юридически важные формулировки сохранены
- [ ] Формат сохранён (заголовки, списки, таблицы)
- [ ] Два документа готовы: А (текст) + Б (отчёт)
