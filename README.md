# ilyakhov-copywriting-skill

## Informational Style Skill for Russian Copywriting

Based on "Pishi, Sokrashchay" (Write, Shorten) by Maxim Ilyakhov & Lyudmila Sarycheva (2nd ed., 2017).

### What it does
Rewrites Russian text in informational style: removes water, officialese, cliches, evaluations.
Makes text brief, clear, and convincing.

### Dual Output (always)
- **Document A**: Rewritten text with preserved formatting
- **Document B**: Change report with tables
